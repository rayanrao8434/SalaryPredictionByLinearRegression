# Salary Prediction using Linear Regression

## Project Overview

This project implements a **Simple Linear Regression** model to predict employee salaries based on their years of experience.

The model is built using Python and the `scikit-learn` machine learning library. The project also includes data visualization, model evaluation, and residual diagnostics.

---

## Objective

The objective of this project is to:

- Analyze the relationship between **Years of Experience** and **Salary**
- Build a predictive machine learning model
- Evaluate model accuracy using regression metrics
- Visualize prediction performance and residual distribution

---

## Dataset Information

The dataset contains salary records for employees with different levels of experience.

### Features

| Feature | Description |
|---|---|
| YearsExperience | Employee experience in years |
| Salary | Employee annual salary |
